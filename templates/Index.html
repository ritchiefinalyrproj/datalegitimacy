Data Legitimacy
using ML

Data sharing is an evolved phenomena which is hard to control. If the information reaching the end user is irrelevant to reality, the sharing must be limited at the root level (user level).

Since predictions are nearly inevitable at the moment, our goal is to find the potential of an information to be non-legit or not. If any Potentially Unwanted Information is detected, the user is informed immediately, thereby inceasing user awareness.
